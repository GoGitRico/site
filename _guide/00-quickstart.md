## Install the AWS command line tools

  `brew install convox`

## Install the AWS command line tools

`brew install awscli`

## Install Convox into your AWS environment

`convox install`

## Set up Convox in your app

`cd ~/myapp`
`convox init`

## Check we're ready & let's go!!

`convox doctor`
`convox start`
